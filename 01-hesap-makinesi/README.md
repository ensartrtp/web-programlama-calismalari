

#  HTML/JS Hesap Makinesi

Bu proje, HTML, CSS ve JavaScript kullanılarak oluşturulmuş, temel aritmetik işlemleri gerçekleştiren basit bir web hesap makinesidir.

## ✨ Özellikler

* **Dört İşlem:** Toplama, çıkarma, çarpma, bölme.
* **Renkli Arayüz:** Fonksiyonel gruplara göre renklendirilmiş butonlar.
* **Responsive Düzen:** CSS Grid ile hizalanmış buton yapısı.
